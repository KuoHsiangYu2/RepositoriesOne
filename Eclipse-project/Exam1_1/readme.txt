電腦作業系統：Windows 10 家用版
Java jdk version：1.8.0_202
資料庫：Microsoft SQL Server 2017
開發IDE：Eclipse IDE for Enterprise Java Developers. Version: 2019-03 (4.11.0)

1.
滑鼠右鍵打開 open.bat，出現命令提示字元視窗。

2.
輸入 run1.bat 開始執行 SQL Server command指令。建立好 Database 跟 Table。

3.
接著再用 Eclipse IDE 打開專案後，啟動Tomcat server開始運行 Web Application。
